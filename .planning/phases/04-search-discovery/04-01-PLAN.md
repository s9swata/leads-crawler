# Phase 4: Search Discovery - Plan 04-01

**Created:** 2026-04-03  
**Phase:** Search & Discovery  
**Confidence:** HIGH

## Goal

Implement Serper.dev Google Search integration to discover leads. Users can search for businesses (e.g., "web development agency Austin") and the tool returns URLs for subsequent scraping.

## Tasks

### Task 1: Create Search Module Structure
Create `src/search/` directory with adapter pattern:

```
src/search/
├── __init__.py
├── adapters/
│   ├── __init__.py
│   ├── base.py          # SearchAdapter ABC
│   └── serper.py        # SerperAdapter implementation
└── models.py             # SearchResult model
```

### Task 2: Implement SearchResult Model
Create `src/search/models.py`:
```python
from pydantic import BaseModel

class SearchResult(BaseModel):
    title: str
    url: str
    snippet: str
```

### Task 3: Implement SerperAdapter
Create `src/search/adapters/serper.py`:
- Use httpx (already in dependencies)
- POST to `https://google.serper.dev/search`
- Include `X-API-Key` header
- Parse `organic` results into `SearchResult` list

### Task 4: Update Search Command
Update `src/cli/commands.py` search command to:
- Check for `SERPER_API_KEY` in settings
- Call `SerperAdapter.search(query, limit)`
- Display results or optionally ingest to database
- Handle API errors gracefully

## Verification Criteria

1. **Requirement 1:** `search --query "web agency"` returns URLs without error
2. **Requirement 2:** API key validation shows helpful error if not set
3. **Requirement 3:** Results integrate with existing `LeadIngestionService`
