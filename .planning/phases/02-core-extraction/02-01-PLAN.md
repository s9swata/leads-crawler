    **social.py** - SocialExtractor:
    - LinkedIn pattern: r'linkedin\.com/(?:in|company|pub)/[a-zA-Z0-9_-]+'
    - Twitter/X pattern: r'(?:twitter\.com|x\.com)/[a-zA-Z0-9_]+'
    - extract_from_html(html) -> dict[str, list[str]] with 'linkedin' and 'twitter' keys

    Use lxml parser for BeautifulSoup. Remove script/style tags before extraction.
  </action>
